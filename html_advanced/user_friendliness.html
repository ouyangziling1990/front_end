本文是一个关于用户友好界面思考的记录

需求如下：

  • 在一个form表单中，需要一个录入调度站的地方，并且能够把调度站的顺序进行排序。
  • 有些调度站需要加入到轮播中，有些不需要加入到轮播中。这个需求不是太明确

实现方式1： jquery插件sort

  1. 北负一西
  2. 北负一东
  3. 北负二
  4. 南广场

实现方式2

    轮播调度站
  1. 北负一西
  2. 北负一东
  3. 北负二
    4. 未轮播调度站
  4. 南广场

两种实现方式的比较

  • 实现方式1以插件方式进行，具有很好的扩展性。但是表示未轮播调度站则没有办法
  • 实现方式2以自写代码实现，具体结构比较清晰，需要一些图标作为表征
  • 用户友好界面以不需要进行培训为最佳，但是也需要考虑到实现的复杂性
